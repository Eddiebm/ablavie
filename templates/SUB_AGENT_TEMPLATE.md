## Spawning Protocol

1. **Identify Need**: Gap in capabilities identified by executive
2. **Define Role**: Create SOUL.md using this template
3. **Configure Access**: Grant necessary API permissions
4. **Onboard**: First-day orientation with parent executive
5. **Train**: Give 3 sample tasks to calibrate quality
6. **Launch**: Full autonomy within defined scope
7. **Review**: 30-day performance review with AblaVie
